Flex Box

display: flex; - inicia o flex no container;
flex-grow: (1 - infinito) - da tamanho ao elemento preechendo os espaços vazios;
flex-shrink: (1 - infinito) - o contrário do grow;
flex-basis: 300px - tamanho inicial do elemento, e o resto é preenchido pelo grow;
flex shorthand - grow / shrink / basis;
flex-direction: (column ou row) - muda a direção que os elementos estão;
flex-order: muda a ordem dos elementos individualmente;
justify-content: (orizontal) - alinha os elementos no eixo principal;
align-item: (vertical) - alinha os elementos no eixo secundárop;
align-self: () - alinha o elemento em particular;
